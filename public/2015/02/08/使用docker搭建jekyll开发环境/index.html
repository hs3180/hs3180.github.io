使用docker搭建jekyll开发环境

2015/02/08

grahamc提供了一个非常简单易用的jekyll开发容器grahamc/jekyll

在一台装有docker的机器上，通过设定如下别名，我们就能在不污染环境的情况下，使用jekyll了

alias jekyll='sudo docker run --rm -v "$PWD:/src" -p 4000:4000 grahamc/jekyll'
